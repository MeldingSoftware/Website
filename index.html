New PC Setup

Windows

*REQUIRES INTERNET*

Wi-Fi Info

Windows

Get a list of Wi-Fi SSID names and passwords that have been used on the computer. Then saves it in a txt file in the script directory for an easy get away. ;)

Update Fix

Windows

For when you're having issues with downloading Windows Updates. If you still have issues with Windows Updates, try to run the DISM and SFC script. And if you still have issues, you may need to manually download and install the update.

Print Spooler Fix

Windows

When your printer gets stuck and doesn't print. Clears the print spooler.

Installed Apps

Windows

Create a list of the installed Apps and their information. Saves the data in a folder on the Desktop. Creates 2 sets of files. One can be opened in Excel (nicer format), and the other can be opened with Notepad (not everyone has Excel). Useful if you need to reinstall your software, you'll have a list.

Largest Files

Windows

Creates a list of the top 10 largest files on your device and saves it to the script location. Figure out what's taking up all your storage space.

Remove Bloatware

Windows

Removes the bloatware that comes with Windows. This is included in the New PC Setup script.

Group Policy Editor

Windows

Only comes with Windows 10/11 Pro or Enterprise editions. Use this script to get Group Policy Editor for Windows 10/11 Home.
