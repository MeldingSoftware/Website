# Melding Software Website

Official website for **Melding Software**.

🌐 https://meldingsoftware.com

## About

Melding Software provides free and open-source Windows and macOS scripts for PC maintenance, repairs, cleanup, diagnostics, software tools, and terminal utilities.

## Repository Contents

- `index.html` — Main website page
- `styles.css` — Website styling and responsive layout
- `javascript.js` — Website interactivity
- `images/` — Website images, logos, QR codes, and visual assets
- `CNAME` — Custom domain configuration for GitHub Pages

## Features

- Windows script downloads
- macOS script install instructions
- Responsive design for desktop and mobile
- Donation/support section
- Gallery previews
- Contact and legal sections
